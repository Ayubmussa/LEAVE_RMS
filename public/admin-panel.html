Weekends (Saturday/Sunday) are automatically excluded and cannot be selected for dining menus.
When enabled, this menu will automatically be applied to all future occurrences of the same day of the week (e.g., if you create a Monday menu, it will apply to all future Mondays).
This will create menus for the specified number of weeks ahead, skipping holidays and weekends.

Upload Holiday Schedule

×
Supported formats: CSV, Excel (.xlsx), Word (.docx). Max size: 10MB
⚠️ Note: Uploading a new file will overwrite all existing holidays for the selected year.

CSV Format:

Date,DayOfWeek,HolidayName
2025-01-01,Wednesday,New Year's Day
2025-01-15,Wednesday,University Closure
1st March,Saturday,Spring Break
15th April,Tuesday,University Closure

Supported Date Formats:

  • YYYY-MM-DD (e.g., 2025-03-01)
  • MM/DD/YYYY (e.g., 03/01/2025)
  • DD/MM/YYYY (e.g., 01/03/2025)
  • Day Month (e.g., 1st March, 15th April)
